each

$.each(T[] array, (T element)-> {} )
$.each(Collection<T> col, (T element)-> {} )
$.each(Map<K,V> map, (K key, V value)-> {} )
$.each(Map<K,V> map, (K key, V value, Integer index)-> {} )
$.each(T[] array, (T element, Integer index)-> {} )
$.each(Collection<T> col, (T element, Integer index)-> {} )
$.each(T[] array, (T element, Integer index, NEXT next)-> {} )
$.each(Collection<T> col, (T element, Integer index, NEXT next)-> {} )

In simpliest case it iterates over a list or array of elements, yielding each in turn to an visitor function. This function will not create a copy of passed parameter and will use instance for iterating, so it's possible for you to get modification exceptions for objects with iterator. Second implementation is useful when you need to also access index of element in iterated array. And the last one has additional parameter NEXT next which could be used by you to stop iterating. You will need to call next.call() on each step in order to proceed with iteration, if next.call() is not called iterator will think that iteration should be stopped.

$.each(new Integer[]{1,2,3}, (Integer i) -> System.out.println(i) );
List<String> list = new ArrayList<String>(){{add("a");add("b");add("c");}};
$.each(list , (String s, Integer i) -> System.out.println(s + " goes on " + i + " place");

Integer wantToFind = 0;
$O<Integer> foundOnIndex = new $O<Integer>(0);
$.each(new Integer[]{1,2,3}, (Integer e, Integer i, NEXT next) -> {
    if (e!=wantToFind) {
        next.call();
    }
    foundOnIndex.set(i);
} 
System.out.println(foundOnIndex.get());

In last example $O wrapper is used to avoid immutable constraint of Integer and insert it into Lambda block.

map

E[] result = $.map(T[] array, (T element)-> {return element.toE()} )
E[] result = $.map(Collection<T> col, (T element)-> {return element.toE()} )
E[] result = $.map(Map<K,V> map, (K key, V value)-> {return element.toE()} )
E[] result = $.map(Map<K,V> map, (K key, V value, Integer index)-> {return element.toE()} )
E[] result = $.map(T[] array, (T element, Integer index)-> {return element.toE()} )
E[] result = $.map(Collection<T> col, (T element, Integer index)-> {return element.toE()} )

Produces a new array of values by mapping each value in list through a transformation function (iterator). If the native map method exists, it will be used instead. If list is a Map, function arguments will be arguments will be (value, key). Map function internally will create new array, will use each to iterate through passed object, and use returned data to fill result.
